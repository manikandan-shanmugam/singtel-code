# singtel-code

This assignment is to find out the animal's behavior.

```
                                      Animal(fly,sing,walk,swim)
                                        |
                                        |
                                        |
                       |-------------|----------------|----------------|---------|
                       |             |                |                |         |   
                       |             |                |                |         |  
                     Birds          Fish             Dolphin         Dog      Cat
                     |==>(f,si,w)    |==>(sw)         |==>(sw)        |=>(!f)   |=>(!f)
                     |=>Duck(sw)     |=>Shark
                     |=>Chicken(!f)  |=>clownfish
                     |=>Rooster(!f)  |=> frog
                     |=>Parrot
                     |=>butterfly(!si)
                     
                     f=fly
                     si=sing
                     w=wlak
                     sw=swim
```
